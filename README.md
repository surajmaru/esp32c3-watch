# esp32c3-watch

## So i'll just directly mention the components i used.

- ESP32 C3 super mini
- 1.3inch oled display
- Tp4056 charging module
- 3.7v lipo battery
- Push button
- Toggle switch
- Touch sensor module (TTP233)
- Watch strap
- Universal PCB board
